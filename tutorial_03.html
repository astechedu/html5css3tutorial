CSS3: New Features

New CSS3 Features:- ------------------ 1. Introduction to CSS3: CSS3 version introduced new features that allowed developers to style HTML elements with less CSS code. 2. The Difference Between CSS and CSS3 The main difference between CSS and CSS3 is that CSS3 offers new features and separates CSS into convenient modules. CSS3 vs CSS: CSS3 does not deprecate older CSS code because it is only an addition to the features offered by CSS1. New Features:- ------------- Box Shadow Opacity Rounded Corners Attribute Selectors New Colors More than Web-Safe Fonts Look at this list of CSS3 tutorials explaining the majority of modules: User Interface Box Model Selectors Backgrounds Borders Media Queries Transforms Text Effects Multiple Column Layout Animations Transition Box Shadow: ---------- One of the CSS3 new features is the box-shadow property that adds a shadow to an element. Example: div { box-shadow: 15px 15px 10px #5196f5; } -------------------------------------------------------------------------------------------- Opacity: -------- One of the CSS3 properties called opacity makes elements see-through or completely transparent. Example: div { opacity: 0.6; } ----------------------------------------------------------------------------------------------------- Rounded Corners: --------------- Before the release of CSS3, developers had to write long code to produce rounded corners. Now, it is enough to apply the border-radius CSS3 property to HTML elements. Example: div { border: 5px solid; border-radius: 50px; } ------------------------------------------------------------------------------------------------------- Attribute Selectors: -------------------- CSS3 also introduced new selectors in addition to the ones in CSS2. Instead of applying IDs or classes for styling, developers can select HTML elements according to their attributes. Example: p { text-align: left; color: blue; } ----------------------------------------------------------------------------------------------------------- New Colors: ---------- One of the CSS3 features is the addition of new colors: RGBA HSL HSLA Gradient Colors There are 4 value types for CSS colors: Color names — blue RGB values — rgb(0, 0, 250) HEX values — #0000FF HSL values — hsl(240, 90%, 78%) Example: #exmp-b {background-color: rgb(0, 0, 255);} /* blue */ #exmp-r {background-color: rgb(255, 0, 0);} /* red */ #exmp-g {background-color: rgb(0, 255, 0);} /* green */ #exmp-b {background-color: rgba(0, 0, 255, 0.4);} /* blue with 0.4 opacity */ #exmp-r {background-color: rgba(255, 0, 0, 0.4);} /* red with 0.4 opacity */ #exmp-g {background-color: rgba(0, 255, 0, 0.4);} /* green with 0.4 opacity */ #exmp-pb {background-color: hsl(240, 60%, 70%);} /* pastel blue */ #exmp-db {background-color: hsl(240, 100%, 25%);} /* dark blue */ #exmp-lb {background-color: hsl(240, 100%, 75%);} /* light blue */ #exmp-b {background-color: hsl(240, 100%, 50%);} /* blue */ #exmp-pb {background-color: hsla(240, 60%, 70%, 0.4);} /* pastel blue with 0.4 opacity */ #exmp-db {background-color: hsla(240, 100%, 25%, 0.4);} /* dark blue with 0.4 opacity */ #exmp-lb {background-color: hsla(240, 100%, 75%, 0.4);} /* light blue with 0.4 opacity */ #exmp-b {background-color: hsla(240, 100%, 50%, 0.4);} /* blue with 0.4 opacity */ ---------------------------------------------------------------------------------------------
