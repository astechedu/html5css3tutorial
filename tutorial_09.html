Box Shadow

box-shadow Property: Add shadows to different elements:
Syntax: box-shadow: none|h-offset v-offset blur spread color |inset|initial|inherit;

#box1 { box-shadow: 5px 10px; }

#box2 { box-shadow: 5px 10px #888888; }

Add a blur effect to the shadow:
#box3 { box-shadow: 10px 10px 8px #888888; }

Define the spread radius of the shadow:
#box4 { box-shadow: 10px 10px 8px 10px #888888; }

Define multiple shadows:
#box5 { box-shadow: 5px 5px blue, 10px 10px red, 15px 15px green; }

Add the inset keyword:
#box6 { box-shadow: 5px 10px inset; }
